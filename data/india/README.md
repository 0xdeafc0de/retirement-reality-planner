# India Historical Data

`nifty50_tri_cpi_annual.csv` combines:

- Annual NIFTY 50 Total Returns Index returns calculated from NSE Indices daily
  Total Returns Index values:
  https://www.niftyindices.com/reports/historical-data
- India annual CPI inflation from the World Bank indicator
  `FP.CPI.TOTL.ZG`.
  https://data.worldbank.org/indicator/FP.CPI.TOTL.ZG?locations=IN

MoSPI also provides a CPI API, but full-data access requires signup and a short
lived access token. World Bank annual inflation is used here so the committed
dataset can be regenerated without credentials.

The CSV is generated by:

```bash
python3 tools/build_india_history.py
```

Rates are stored as percentages because the replay loader accepts either
percentages such as `8.5` or decimals such as `0.085`.
